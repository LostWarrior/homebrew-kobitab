# homebrew-kobitab

Homebrew tap for KobiTab.

## Install

```bash
brew tap LostWarrior/kobitab
brew install kobitab
```

## Upgrade

```bash
brew update
brew upgrade kobitab
```

## Uninstall

```bash
brew uninstall kobitab
brew untap LostWarrior/kobitab
```

## Notes

- Tap cask file: `Casks/kobitab.rb`
- Binaries are published at: https://github.com/LostWarrior/Kobitab/releases/latest
